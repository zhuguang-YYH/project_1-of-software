# 数据库初始化 - 快速指南

## 🎯 3分钟快速开始

### 步骤 1️⃣：部署初始化函数（2分钟）
```
在微信开发者工具中：
1. 点击左侧 "云函数"
2. 右键 "init_db" 文件夹
3. 选择 "上传并部署" → "云端安装依赖"
4. 等待完成
```

### 步骤 2️⃣：执行初始化（1分钟）
```
在小程序控制台输入：

wx.cloud.callFunction({
  name: 'init_db',
  data: {}
}).then(res => console.log('成功:', res))
  .catch(err => console.error('失败:', err));
```

### 步骤 3️⃣：验证结果（多个方式）
- ✅ 打开首页 → 应该看到排行榜数据
- ✅ 打开活动页 → 应该看到活动列表
- ✅ 打开兑换页 → 应该看到商品列表
- ✅ 打开借阅页 → 应该看到物资列表

## ⚡ 关键信息

**初始化函数位置**：
```
cloudfunctions/
└── init_db/
    ├── index.js
    └── package.json
```

**执行时间**：< 5秒

**安全性**：✅ 不会重复创建已存在的数据

**可恢复**：✅ 可随时删除集合后重新初始化

## 🔍 检查列表

部署前检查：
- [ ] 云函数已上传到腾讯云
- [ ] 云环境 `cloud1` 确实存在
- [ ] 小程序能连接到云环境

部署后检查：
- [ ] init_db 函数执行成功（返回 code:0）
- [ ] 可以看到初始化日志
- [ ] 数据库集合已创建
- [ ] 小程序首页能显示数据

---

## 💡 提示

如果出错，查看：
1. **云函数日志** - 在腾讯云平台查看完整日志
2. **浏览器控制台** - 查看调用结果
3. **数据库** - 检查集合是否存在
4. **网络连接** - 确保能连接到云环境
