points.js (160行) - 积分管理

├── hasEnoughPoints() - 足额检查

borrow.js (200行) - 物资借阅
├── getItems() - 物品列表

activity.js (210行) - 活动管理

## 🚀 快速开始指南

### 第一步：打开项目
```bash
微信开发者工具 → 打开项目 → D:\WeChatApp
```

### 第二步：编译检查
```
点击【编译】按钮 → 无编译错误 ✅
```

### 第三步：导入服务
```javascript
const puzzleService = require('../../services/puzzle.js');
const rankingService = require('../../services/ranking.js');
```

### 第四步：调用API
```javascript
// 获取每日谜题
const puzzle = await puzzleService.getTodayPuzzle();

// 获取排行榜
const ranking = await rankingService.getFullRanking({ page: 1 });

// 提交答题
const result = await puzzleService.submitAnswer(puzzleId, optionId);
```
