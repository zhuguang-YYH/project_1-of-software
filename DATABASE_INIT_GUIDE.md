## 🚀 部署和执行步骤

### 方案 A：使用微信开发者工具（推荐）

#### 第1步：部署初始化函数
1. 打开微信开发者工具
2. 在左侧找到 **云函数** 栏目
3. 找到 `init_db` 文件夹，右键选择 **上传并部署** → **云端安装依赖**
4. 等待部署完成

#### 第2步：执行初始化
1. 打开任意页面（如首页）
2. 在控制台（Console）输入以下代码：

```javascript
wx.cloud.callFunction({
  name: 'init_db',
  data: {},
  success: res => {
    console.log('初始化结果:', res);
  },
  fail: err => {
    console.error('初始化失败:', err);
  }
});
```

3. 执行，观察返回结果
4. 应该看到 `{ code: 0, message: '数据库初始化成功' }`

### 方案 B：腾讯云开发平台

#### 第1步：进入云函数控制台
1. 访问 https://console.cloudbase.tcb.com/
2. 切换到你的环境 `cloud1`
3. 点击左侧 **云函数**
4. 点击 **新建云函数** 按钮

#### 第2步：创建函数
1. 函数名填写：`init_db`
2. 运行环境选择：`Node.js 12`
3. 上传 `cloudfunctions/init_db/` 目录中的文件
4. 点击 **保存**

#### 第3步：执行函数
1. 找到 `init_db` 函数
2. 点击 **测试** 按钮
3. 事件模板选择空对象 `{}`
4. 点击 **执行**
5. 查看输出结果

## ✅ 验证初始化是否成功

初始化完成后，在腾讯云开发平台的 **数据库** 栏目中：

1. 应该能看到上述所有集合已创建
2. 各集合中应该有对应的测试数据
3. 可以打开小程序，查看首页是否能显示数据

示例验证步骤：
1. 打开 **排行榜** 页面 → 应该看到 3 个用户的排行信息
2. 打开 **活动** 页面 → 应该看到 2 个活动
3. 打开 **物资借阅** 页面 → 应该看到 4 个物资
4. 打开 **积分兑换** 页面 → 应该看到 5 个商品
5. 打开 **DUD聊天** 页面 → 输入"你好" → 应该收到回复

## 🔄 重新初始化

如果需要清空所有数据并重新初始化：

1. 进入腾讯云开发平台
2. 在 **数据库** 栏目删除各个集合
3. 重新执行初始化函数

**注意**：删除集合是不可逆的操作，请谨慎！

## 🛠️ 常见问题

### 问题 1：执行失败 "Function not found"
**原因**：init_db 函数未部署
**解决**：按照上述步骤重新部署 init_db 函数

### 问题 2：执行成功但数据没有导入
**原因**：可能是权限问题
**解决**：
- 检查云函数执行日志（在云平台查看）
- 确保有数据库写入权限
- 查看集合是否已创建

### 问题 3：只有部分数据导入
**原因**：某个集合创建失败
**解决**：
- 检查控制台日志（console.log 输出）
- 查看是否有重复的关键字约束
- 手动创建缺失的集合

### 问题 4：如何修改初始数据？
**解决**：
1. 编辑 `cloudfunctions/init_db/index.js` 中的数据
2. 重新上传并部署
3. 清空集合后重新运行初始化
