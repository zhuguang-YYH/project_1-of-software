# 配置依赖说明

本项目运行依赖以下配置文件和目录。

## 基本配置

- `project.config.json`
  - 微信开发者工具的项目公共配置。
  - 指定小程序源码目录 `miniprogram/`、云函数目录 `cloudfunctions/`、AppID、基础库版本、云环境等。

- `database_security_rules.json`
  - 云数据库安全规则配置。
  - 小组成员部署或迁移云环境时，可参考该文件同步数据库权限规则。

- `miniprogram/config/index.js`
  - 小程序运行时配置。
  - 包含云环境 ID、AppID、云函数 action 名称、超时时间、业务状态和错误码等。

- `cloudfunctions/*/package.json`
  - 各云函数的依赖声明。
  - 当前云函数主要依赖 `wx-server-sdk`。

## 开发者本地配置

- `project.private.config.json`
  - 这是微信开发者工具生成的本地私有配置。
  - 不同成员的本地设置可能不同，因此不提交。
  - pull 项目后，用微信开发者工具打开项目时会自动生成自己的 `project.private.config.json`。

## 目录依赖

- `miniprogram/`
  - 小程序前端源码和页面资源。

- `cloudfunctions/`
  - 云函数源码及各云函数自己的配置、依赖声明。

- `tests/`
  - 自动化测试代码。
  - 测试入口为 `tests/run-tests.js`。

## 环境依赖

- 微信开发者工具
- Node.js
- 微信云开发环境权限

## 注意事项

- 如果更换 AppID 或云环境，需要同步修改 `project.config.json` 和 `miniprogram/config/index.js`。
- 如果修改云数据库权限，需要同步更新 `database_security_rules.json`。
- 如果修改云函数依赖，需要更新对应云函数目录下的 `package.json`。
